Sobre Mim

Apresentação - Robson Pepplow Lopes

Sou Robson Pepplow Lopes, um desenvolvedor full-stack apaixonado por programação. Tenho 39 anos, sou pai de 3 filhos e casado há 17 anos. Nos últimos anos, dediquei-me inteiramente à área de desenvolvimento, buscando constantemente aprimorar minhas habilidades e conhecimentos em tecnologias como Node.js, React.js, Vue.js, Quasar, Python e frameworks como Nest.js para construção de backends e APIs com Firebase e MongoDB.

Há cerca de um ano, iniciei minha jornada no desenvolvimento web ao me inscrever em uma renomada Escola de Desenvolvimento Web. Durante o curso, adquiri conhecimentos essenciais em JavaScript, abrangendo tópicos como variáveis, constantes, operadores, estruturas condicionais, arrays, strings e funções. Embora, na época, não pudesse prosseguir com a oportunidade devido a outros compromissos, encontrei minha própria maneira de estudar com a ajuda de um amigo e mentor, Erick Amarante, que me orientou e abriu portas para projetos empolgantes, como o Actflow.

Atualmente, estou cursando o 2º ano de Tecnologia da Informação na Faculdade Estácio, buscando aprimorar ainda mais meus conhecimentos e obter uma formação sólida. Além disso, continuo estudando e focando em aperfeiçoar minhas habilidades em React, Vue.js e iniciando em Typescript.

Nos últimos 5 meses, atuei como freelancer na startup Othree, onde desenvolvi o frontend com o framework Quasar e o backend utilizando Nest.js e o Firebase como banco de dados. Também tenho experiência com automação e extração de dados, utilizando o Puppeteer do Node.js em conjunto com o Python e o Selenium.

Sou uma pessoa dedicada e flexível, facilmente me adaptando a mudanças e desafios. Minha paixão é desenvolver projetos web e estou animado para aplicar meu conhecimento em projetos desafiadores e impactantes. Estou em busca de novas oportunidades e emprego na área de desenvolvimento, prontamente disposto a contribuir com o sucesso da empresa.

Projetos Realizados

Projeto Dashboard Actflow - Gestão Contábil

Descrição do Projeto

Este projeto foi desenvolvido para a startup Othree Tech Workspace/Act-Flow, com o objetivo de automatizar os processos usando bots com Node.js e Puppeteer para raspar dados, agendar e baixar notas fiscais de prefeituras, utilizando Nest.js e APIs para integração com o banco de dados Firebase, também com API REST através do Nest.js. Inicialmente, o frontend foi desenvolvido com Vue.js, conforme mostrado nos slides abaixo, e futuramente será migrado para o framework Quasar.

Essa abordagem otimizou significativamente o tempo de desenvolvimento, uma vez que a documentação possui diversos exemplos que facilitam a criação do frontend. Além disso, a gestão de todos os dados raspados, como processos de pagamentos de guias em instituições como prefeituras e outros órgãos, é realizada para os clientes de escritórios de contabilidade devidamente cadastrados no sistema da Actflow - Gestão Contábil.

Para mais informações sobre o projeto, entre em contato diretamente com Erick Amarante, um dos sócios da Empresa, pelo telefone (41) 99793-9726.

Neste projeto, atuo como freelancer, onde recebo pacotes de trabalho para desenvolvimento e entrega. Estou trabalhando nele desde 06/2006 e foi o meu primeiro contato com desenvolvimento em equipe, o que me permitiu aprimorar minhas habilidades técnicas e sociais, atuando em todas as stacks mencionadas acima.

Projeto Sistema Gestão De Locações de Trajes

Este projeto foi desenvolvido para a loja de locação de trajes da minha irmã, que já está no mercado há 10 anos. Anteriormente, ela utilizava planilhas do Excel para gerenciar os trajes locados e os contratos. Quando adquiri experiência na área, me propus a criar uma aplicação para controlar e gerenciar todas essas informações, que antes eram manuseadas no Excel.

Realizei a migração das tabelas para o Firebase e criei uma API no Nest.js para manipulá-las. No frontend, utilizei o framework Quasar para desenvolver a interface da aplicação. Ainda estou em constante desenvolvimento, aprimorando o projeto continuamente.
